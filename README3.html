# Basic writing and formatting syntax Headings To create a heading, add one to six # symbols before your heading text. The number of # you use will determine the size of the heading. // # this is The largest heading // ## The second largest heading // ###### The smallest heading the output will look like something like the below: this is The largest heading The second largest heading The smallest heading Quoting text You can quote text with a >. In the words of Abraham Lincoln: // > Pardon my French Output In the words of Abraham Lincoln: > Pardon my French Links You can create an inline link by wrapping link text in brackets [ ], and then wrapping the URL in parentheses ( ). You can also use the keyboard shortcut command + k to create a link. This site was built using [GitHub Pages](https://pages.github.com/). Content attachments To see content attachments, you must have a GitHub App that uses the Content Attachments API installed on the repository. For more information. Content attachments will not be displayed for URLs that are part of a markdown link.
